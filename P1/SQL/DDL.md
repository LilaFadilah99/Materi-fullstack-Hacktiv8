# `DDL`

`DDL` adalah jenis query yang bertugas untuk pembuatan / modifikasi / penghapusan `database`, `table`, dan `kolom`.

### `Contoh`
1. CREATE DATABASE
2. DROP DATABASE
3. CREATE TABLE
4. DROP TABLE
5. etc...

### `NOTE`
Jika ada relasi antar table, pastikan table yang menjadi referensi dibuat terlebih dahulu.
