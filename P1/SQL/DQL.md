# `DQL`

`DQL` adalah jenis query yang bertugas untuk manipulasi bentuk `data` hasil `SELECT` dengan lebih dinamis.

### `Contoh`
1. AS
2. DISTINCT
3. JOIN
4. GROUP BY
5. ORDER BY
6. HAVING
7. LIMIT
8. LIKE
9. etc..

### `CONTOH Aggregate Function in SQL`
1. SUM()
2. AVG()
3. COUNT()

### `Urutan DQL dalam proses SELECT`

FROM & JOIN

WHERE

GROUP BY

HAVING

ORDER BY

LIMIT
